# Flutter Mouse Robot
Aplicativo que se conecta ao [client](https://github.com/diegorkalschne/mouse_robot_client_desktop) feito em `Node.js` e `robotjs` para controlar o mouse do computador utilizando o celular.

## Debug
1. `git clone`
2. `flutter pub get`
3. `flutter run`
